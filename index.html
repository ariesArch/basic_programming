Image Tag:

use img tag to show image document on web page. There is one essential attribute, src(source) which load image from the source. There is another optional attribute , alt(alternative) which describe the text if the image cannot be loaded.

Anchor Tag:

Link between pages and elements. It has important attribute "href" which refer to redirectable location. Another attribute "target" configure the browser tag to open link in another new tab or current tab.
